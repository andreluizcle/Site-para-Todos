Este é o repositório do site oficial do **Colégio Recreando**. O projeto foi desenvolvido para fornecer informações claras e acessíveis a pais, alunos e colaboradores, servindo como uma vitrine digital para a proposta pedagógica da escola.

🔗 **Acesse o site:** [colegiorecreando.com.br](https://colegiorecreando.com.br)

---

## 📝 Sobre o Projeto

O site tem como objetivo centralizar as informações da instituição, facilitando o contato com a secretaria e apresentando a infraestrutura e os diferenciais do colégio.

### ✨ Funcionalidades Principais
* **Página Inicial:** Destaques e visão geral da escola.
* **Proposta Pedagógica:** Detalhes sobre a metodologia de ensino.
* **Galeria:** Fotos das instalações e eventos.
* **Contato:** Formulário integrado e link direto para WhatsApp.
* **Localização:** Integração com Google Maps para facilitar o acesso.
